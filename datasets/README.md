# Open Datasets for EcareBots

This directory contains curated information about open-source datasets relevant to healthcare AI, multi-modal accessibility, and elderly care.

## Directory Structure

- **open-datasets.md** - Comprehensive catalog of publicly available datasets with descriptions, licenses, and use cases

## Usage Guidelines

1. **Verify licenses** before using any dataset (ensure commercial use is permitted)
2. **Check data quality** (completeness, accuracy, recency)
3. **Respect privacy** (use only ethically sourced, de-identified data)
4. **Cite sources** (provide attribution as required by licenses)
